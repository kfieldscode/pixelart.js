# Pixel Art Maker JS

My first Javascript project from early 2019! A pixel-art maker in Javascript, made with pure Javascript, CSS, and HTML.

Features Included:
- Pick any color
- Drag for non-click coloring
- Double-click for clear
- Clear entire canvas
- Paint entire canvas
